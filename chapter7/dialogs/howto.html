Verwendungshinweise zur Tartan-Erstellung

Wählen Sie für jede Farbe, die Ihr Tartan enthalten soll, eine Farbe und eine Stichanzahl. Die Stichanzahl kann ein Wert zwischen 2 (so viele muss ein traditioneller Weber bei einem 2-über-2-unter-Webverfahren mindestens einsetzen) und 72 (diese Obergrenze ist willkürlich).

Ein Tartan kann insgesamt bis zu 120 Stichen enthalten.

Denken Sie daran, dass die Stich/Farbe-Kombinationen horizontal und vertikal wiederholt werden. So wird das Tartan-übliche Aussehen gestaltet!

Die Stichzahl muss ein Vielfaches von 2 sein. Auch das geht wieder auf die Bedingungen zurück, unter denen Tartans gewoben werden.
